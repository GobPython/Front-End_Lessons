Домашнее задание после урока 19

Создайте форму. Внутри неё должны находиться следующие поля:
- Инпут для ввода имени, данное поле обязательно должно быть заполнено. Используйте placeholder, чтобы сообщить пользователю, какие данные он должен ввести в данное поле
- Инпут для ввода возраста. Минимальное значение, которое пользователь может ввести в данную форму, равно 6, максимальное - 100. Используйте placeholder, чтобы сообщить пользователю, какие данные он должен ввести в данное поле
- Радиокнопки, где пользователь выбирает своё семейное положение. Данные радиокнопки объедините с помощью <fieldset>
- Инпут, который имеет доступ только для чтения и имеет значение = ‘это значение отправляется с формой’. <textarea>, максимальное количество символов, которое пользователь может ввести в это поле, равно 300
- Добавить кнопку для отправки формы.
