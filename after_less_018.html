Домашнее задание после урока 18

1. Создайте выпадающий список со списком стран, в данном списке должны присутствовать страны с разных континентов. Для того чтобы сгруппировать данные элементы, используйте тег <optgroup>
2. Из данного списка стран пользователь должен выбрать страны, в которых он хочет побывать. Для этого необходимо позволить пользователю делать множественный выбор
3. Необходимо добавить подпись для выпадающего списка с помощью <label>
4. Ниже добавьте поля для текста, где пользователь должен написать, почему он выбрал эти страны. Для данного тега также необходимо добавить подпись, которая сообщает, для чего предназначается данный элемент
5. Все эти элементы должны быть заключены в <form> и иметь кнопку для отправки
